# Decision Tree Classification Using UCI Iris Dataset

A machine learning classification project developed for **Business Intelligence & Analytics**.

This project applies a **Decision Tree Classifier** to predict iris flower species using measurements from the **UCI Iris Dataset**.

---

## Project Objective

The goal of this project is to build and evaluate a Decision Tree Classification model capable of identifying iris flower species based on flower measurements.

The project demonstrates:

* Data acquisition from UCI repository
* Data exploration using Pandas
* Data preprocessing
* Model training and testing
* Performance evaluation
* Decision tree visualization

---

## Dataset

Dataset: **Iris Dataset (UCI Machine Learning Repository)**

Features:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

Target Classes:

* Iris-setosa
* Iris-versicolor
* Iris-virginica

Dataset Size:

* 150 Instances
* 4 Features
* 3 Classes

Dataset Source:
https://archive.ics.uci.edu/ml/datasets/iris

##  Workflow

### 1. Data Acquisition and Exploration

* Load dataset directly from UCI
* Display first records
* Identify features and target

### 2. Model Training and Testing

* Split dataset (80% train / 20% test)
* Train Decision Tree using Gini Index
* Measure model accuracy

### 3. Performance Evaluation

* Generate predictions
* Create confusion matrix
* Calculate accuracy

### 4. Decision Tree Visualization

* Visualize model structure
* Identify root node
* Interpret classification decisions
